# Toxic Comment Classification

Machine Learning and NLP project based on the Jigsaw Toxic Comment Classification Challenge.

## Project Goal

Build a multi-label text classification model capable of identifying:

- Toxic
- Severe Toxic
- Obscene
- Threat
- Insult
- Identity Hate

comments from Wikipedia discussions.

## Dataset

Kaggle Jigsaw Toxic Comment Classification Challenge.
